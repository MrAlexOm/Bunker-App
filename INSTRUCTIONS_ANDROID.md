# Установка Bunker на Android

Bunker — это веб-приложение (PWA), которое можно установить как обычное приложение на телефон. Оно работает офлайн и не требует доступа в интернет после первого открытия.

## Как установить

1. **Открой ссылку** на приложение в браузере **Chrome** на Android (например, https://astonishing-parfait-511816.netlify.app).
2. В правом верхнем углу нажми на **три точки** (меню).
3. Выбери пункт **«Установить приложение»** или **«Добавить на экран "Домой"**.
4. Подтверди установку.

После этого иконка Bunker появится на рабочем столе. Приложение запускается как обычное, работает без интернета.

## Если не получается установить

- Убедись, что используешь **Chrome** (в других браузерах функция может отсутствовать).
- Попробуй обновить браузер до последней версии.
- Если кнопки нет, открой ссылку, подожди полной загрузки, затем закрой браузер и снова открой – иногда установка предлагается автоматически.

## Сборка APK файла (для разработчиков)

Если нужен отдельный APK‑файл, который можно раздавать как установочный:

### Способ 1: EAS Build (рекомендуется)
```bash
eas build --platform android --profile preview
```

### Способ 2: Локальная сборка
```bash
# Сначала установи Java Development Kit (JDK)
# Затем выполни:
npx expo prebuild --platform android
cd android
./gradlew assembleRelease
```

### Способ 3: Expo Go (для тестирования)
```bash
npx expo start --dev-client
```

## Требования для сборки APK

- **Java Development Kit (JDK)** версии 17 или выше
- **Android Studio** с установленными SDK
- **Node.js** версии 18+
- **EAS CLI**: `npm install -g eas-cli`

## Если возникли ошибки при сборке

1. **Установи Java JDK**:
   - Скачай с [Oracle JDK](https://www.oracle.com/java/technologies/downloads/)
   - Установи и добавь в PATH
   - Проверь: `java -version`

2. **Настрой переменные окружения**:
   ```bash
   export JAVA_HOME=/path/to/java
   export ANDROID_HOME=/path/to/android-sdk
   ```

3. **Очисти кэш**:
   ```bash
   npx expo install --fix
   eas build --clear-cache
   ```

## Готовый APK

После успешной сборки APK файл будет находиться в:
```
android/app/build/outputs/apk/release/app-release.apk
```

## Для разработчиков

Если нужна нативная сборка APK (для распространения вне Google Play), используй EAS Build. APK файл можно распространять через Telegram, email или прямые ссылки.
